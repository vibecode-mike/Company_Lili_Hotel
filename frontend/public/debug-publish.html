🔍 发布按钮调试工具

帮助诊断"发布"按钮点击后没有反应的问题

📋 使用说明：
  1. 在主应用中填写完整的活动表单（标题、通知訊息、上传图片等）
  2. 点击"发布"按钮
  3. 立即切换到这个页面
  4. 点击下方的"开始诊断"按钮

第1步：检查服务器连接

第2步：测试简单请求

第3步：查看浏览器控制台

按 F12 打开开发者工具，切换到 Console 标签页

查找以下标记的日志：

  • 🔵 handlePublish called
  • 📤 Starting image upload...
  • 🏗️ Building form data...
  • 🔍 Validating form...
  • 📡 Sending request to backend:
  • ❌ 如果看到红色错误，请记录错误信息

第4步：检查网络请求

在开发者工具中切换到 Network 标签页

  1. 找到 POST /api/v1/campaigns 请求
  2. 查看 Status Code (应该是 200，如果是 400 则有错误)
  3. 点击该请求，查看 Preview 或 Response 标签
  4. 如果有错误，复制完整的响应内容

第5步：常见问题检查

  • 是否填写了活动标题？
  • 是否填写了通知訊息？
  • 是否选择了模板类型（不是"請選擇"）？
  • 是否上传了图片？
  • 如果启用了动作按钮，是否填写了对应的URL/文字？
  • 浏览器是否清除了缓存？(按 Ctrl+Shift+R 硬刷新)
