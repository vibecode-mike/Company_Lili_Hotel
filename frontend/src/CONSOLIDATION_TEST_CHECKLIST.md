## 🎯 關鍵測試場景

### 場景 1：創建觸發關鍵字自動回應
1. [ ] 進入自動回應列表頁面
2. [ ] 點擊「新增」按鈕
3. [ ] 選擇回應類型：「觸發關鍵字」
4. [ ] 添加 3 個關鍵字：「飯店」、「房型」、「預訂」
5. [ ] 輸入訊息文字
6. [ ] 點擊「好友的顯示名稱」按鈕插入變數
7. [ ] 驗證變數顯示為黃色標籤
8. [ ] 添加第二個訊息
9. [ ] 點擊儲存
10. [ ] 驗證成功通知
11. [ ] 驗證返回列表頁面

### 場景 2：創建排程一律回應
1. [ ] 進入自動回應列表頁面
2. [ ] 點擊「新增」按鈕
3. [ ] 選擇回應類型：「一律回應」
4. [ ] 選擇觸發時間：「指定日期或時間」
5. [ ] 設定開始日期和時間
6. [ ] 設定結束日期和時間
7. [ ] 輸入訊息文字
8. [ ] 點擊儲存
9. [ ] 驗證成功通知

### 場景 3：多訊息排序
1. [ ] 創建包含 3 個訊息的自動回應
2. [ ] 點擊第 3 個訊息的「上移」按鈕
3. [ ] 驗證訊息移到第 2 位
4. [ ] 驗證預覽面板順序更新

7. [ ] 點擊中間訊息的「刪除」按鈕
8. [ ] 驗證訊息被刪除
9. [ ] 驗證預覽面板更新

### 場景 4：表單驗證
