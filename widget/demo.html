力麗飯店 AI Chatbot Widget Demo

右下角的聊天按鈕就是嵌入的 widget，點擊即可開始對話。

嵌入方式

<script
  src="https://YOUR_DOMAIN/widget/lili-chatbot.js"
  data-api="https://crmpoc.star-bit.io"
></script>

可選參數

  • data-api — 後端 API 網址（必填）
  • data-position — "right"（預設）或 "left"
  • data-color — 主色 hex，預設 "#0f6beb"
  • data-title — 標題文字，預設 "AI 訂房助手"
  • data-greeting — 初始提示文字
