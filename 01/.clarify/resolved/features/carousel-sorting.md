# 定位
Feature：spec/features/message_template.feature 內 `Rule: 轮播圖卡支援圖片排序調整操作` 與其 Example（約第132-140行）。

# 多選題
| 選項 | 描述 |
|------|------|
| A | 需要排序功能，應提供拖拽或上下箭頭來重新排序輪播圖。 |
| B | 不提供排序，Example 才是正確描述；請刪除/修正該 Rule。 |
| C | 提供簡化排序（例如輸入排序號碼或只允許整體反轉），請說明交互細節。 |
| Short | 其他（<=5字）。 |
