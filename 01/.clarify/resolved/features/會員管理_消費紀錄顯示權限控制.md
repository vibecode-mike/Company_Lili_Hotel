# 釐清問題

消費紀錄（v0.2）的查看權限控制為何？是所有管理員都可查看，還是需要特定權限？

# 定位

Feature：spec/features/member_management.feature Rule 關於消費紀錄（約第180-218行）

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | v0.1 所有管理員都可查看，v0.2 再加入權限控制 |
| B | 僅特定角色（如 superadmin、財務人員）可查看消費紀錄 |
| C | 查看消費紀錄需額外勾選權限，不受角色限制 |
| Short | 其他設計（<=5字）|

# 影響範圍

影響權限系統設計、資料隱私保護、角色定義、功能模組存取控制，以及合規性要求。

# 優先級

Low

---
# 解決記錄

- **回答**：A - v0.1 所有管理員都可查看，v0.2 再加入權限控制
- **更新的規格檔**：spec/features/member_management.feature
- **變更內容**：在「若串接 CRM，可查看該會員消費紀錄」規則中新增「v0.1 所有管理員都可查看消費紀錄」Example，並新增「消費紀錄查看權限控制（v0.2 功能）」規則，定義 v0.2 版本的權限控制機制
