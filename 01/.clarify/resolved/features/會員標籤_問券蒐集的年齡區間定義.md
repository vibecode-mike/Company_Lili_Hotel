# 釐清問題

問券蒐集的年齡區間標籤有哪些？僅「18-25 歲」「26-35 歲」還是有完整的年齡段定義？

# 定位

Feature：spec/features/member_tag_management.feature Rule 關於年齡區間比對（約第94-99行）

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 「18-25」「26-35」「36-45」「46-55」「56-65」「66+」六個區間 |
| B | 「18以下」「18-25」「26-35」「36-50」「51+」五個區間 |
| C | 由管理員自訂年齡區間與標籤名稱 |
| Short | 請提供完整區間（<=5字）|

# 影響範圍

影響問券蒐集選項設計、標籤自動生成邏輯、會員年齡分析報表，以及行銷活動年齡分眾策略。

# 優先級

Medium

---
# 解決記錄

- **回答**：B - 「18以下」「18-25」「26-35」「36-50」「51+」五個區間
- **更新的規格檔**：spec/features/member_tag_management.feature
- **變更內容**：在「問券蒐集標籤支援年齡區間比對」規則中新增完整年齡區間定義（18以下、18-25、26-35、36-50、51+），並補充各區間的具體 Example
