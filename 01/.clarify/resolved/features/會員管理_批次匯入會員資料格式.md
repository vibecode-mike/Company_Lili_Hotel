# 釐清問題

系統是否支援批次匯入會員資料？若支援，接受的檔案格式與欄位映射規則為何？

# 定位

Feature：會員管理 - 批次匯入功能（spec/requirement.md 提到「新增會員」）

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 支援 CSV 格式批次匯入（提供範本下載） |
| B | 支援 Excel 格式批次匯入（.xlsx） |
| C | 支援 CSV 與 Excel 兩種格式 |
| D | 不支援批次匯入（僅支援單筆新增與 PMS 同步） |
| Short | 提供其他簡短答案（<=5 字） |

註：若支援批次匯入，需定義必填欄位、欄位驗證規則、重複資料處理邏輯（覆蓋或跳過）

# 影響範圍

影響會員管理功能範圍、資料匯入介面設計、欄位驗證與錯誤處理機制、以及使用者操作效率

# 優先級

Medium - 影響功能完整性與使用者體驗，但不阻礙核心功能

---

# 解決記錄

- **回答**：C - 支援 CSV 與 Excel 兩種格式
- **更新的規格檔**：spec/features/member_management.feature
- **變更內容**：新增 Rule「支援批次匯入會員資料（CSV 與 Excel 格式）」，包含 4 個 Example：下載匯入範本、CSV 格式匯入、Excel 格式匯入、重複資料處理（依 LINE UID 跳過）
- **解決時間**：2025-11-14
