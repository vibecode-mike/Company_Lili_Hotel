# 釐清問題

「當日零時 00:00 自動登出」的時區基準為何？是伺服器時區還是使用者瀏覽器時區？

# 定位

Feature：spec/features/登入系統.feature Rule 關於自動登出（約第61-63行）

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 使用伺服器時區（台灣 UTC+8），所有使用者統一於同一時間登出 |
| B | 使用使用者瀏覽器時區，各地使用者於當地零時登出 |
| C | v0.1 使用伺服器時區，未來支援時區選擇 |
| Short | 其他設計（<=5字）|

# 影響範圍

影響會話管理邏輯、登出觸發機制、跨時區使用情境、使用者體驗，以及安全性設計。
