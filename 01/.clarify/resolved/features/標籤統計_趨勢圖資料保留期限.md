# 釐清問題

標籤趨勢圖資料（7天/30天/90天）的歷史資料保留期限為何？是否永久保留，還是定期清理舊資料？

# 定位

Feature：spec/features/label_statistics.feature Rule 關於趨勢圖（約第49-76行）

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 永久保留所有歷史資料，支援任意時間範圍查詢 |
| B | 保留最近 12 個月資料，超過自動歸檔或刪除 |
| C | 保留最近 90 天明細資料，超過僅保留每日/每週/每月彙總 |
| Short | 其他策略（<=5字）|

# 影響範圍

影響資料庫儲存容量規劃、查詢效能、資料歸檔策略、備份還原設計，以及歷史分析能力。

# 優先級

Low

---
# 解決記錄

- **回答**：A - 永久保留所有歷史資料，支援任意時間範圍查詢
- **更新的規格檔**：spec/features/label_statistics.feature
- **變更內容**：新增「標籤趨勢圖歷史資料永久保留」規則，補充「永久保留所有歷史資料」與「支援任意時間範圍查詢」的 Example
