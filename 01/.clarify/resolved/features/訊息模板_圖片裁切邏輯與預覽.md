# 釐清問題

非 1:1 比例圖片自動裁切時，裁切邏輯為何？是取中心區域、上方區域，還是提供使用者調整裁切位置？

# 定位

Feature：spec/features/message_template.feature Rule 關於圖片裁切（約第279-286行）

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 自動取中心區域裁切，不提供調整選項 |
| B | 上傳後顯示裁切預覽工具，使用者可拖曳調整裁切位置 |
| C | 提供三種預設裁切位置（上/中/下或左/中/右），使用者選擇 |
| Short | 其他方式（<=5字）|

# 影響範圍

影響圖片上傳 UX、前端裁切工具開發、圖片處理邏輯、使用者滿意度，以及視覺呈現品質。
