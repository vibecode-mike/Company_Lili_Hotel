# 釐清問題

刪除輪播圖卡後，剩餘圖卡的序號如何調整？是自動遞補，還是保留原序號形成跳號？

# 定位

Feature：spec/features/message_template.feature Rule 關於刪除操作（約第150-156行）

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 自動遞補，刪除第 3 張後第 4 張變成第 3 張，序號連續 |
| B | 保留原序號，形成跳號（圖卡 1、2、4、5，缺少 3） |
| C | 重新編號，所有圖卡序號從 1 開始重新排序 |
| Short | 其他邏輯（<=5字）|

# 影響範圍

影響輪播圖卡儲存結構、序號管理邏輯、UI 顯示、資料一致性，以及前端編輯體驗。
