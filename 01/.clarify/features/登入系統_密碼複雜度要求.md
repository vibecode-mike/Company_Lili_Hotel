# 釐清問題

一般登入的密碼複雜度要求為何？最小長度、必須包含大小寫/數字/特殊符號等規則？

# 定位

Feature：spec/features/登入系統.feature Rule 關於帳號密碼（約第10-11行）

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 最小 8 字元，必須包含大小寫英文、數字、特殊符號 |
| B | 最小 8 字元，必須包含英文與數字，特殊符號選填 |
| C | 最小 6 字元，無其他限制（僅禁止弱密碼如 123456） |
| Short | 請提供規則（<=5字）|

# 影響範圍

影響管理員註冊流程、密碼驗證邏輯、資訊安全等級、合規性要求，以及使用者體驗。
