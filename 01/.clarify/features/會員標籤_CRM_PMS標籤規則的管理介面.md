# 釐清問題

CRM/PMS 標籤自動產生規則（如消費金額門檻）的管理介面在哪裡？是在「標籤管理頁」還是「PMS 整合設定頁」？

# 定位

Feature：spec/features/member_tag_integration.feature Rule 關於自訂條件（約第54-84行）

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 在「標籤管理頁」新增「標籤規則」Tab，集中管理所有自動標籤規則 |
| B | 在「PMS 整合設定頁」配置標籤規則，與整合設定綁定 |
| C | 分散管理：會員標籤規則在標籤管理頁，PMS 標籤規則在 PMS 設定頁 |
| Short | 其他位置（<=5字）|

# 影響範圍

影響功能模組組織架構、使用者操作流程、權限控制設計、標籤規則可維護性，以及系統架構清晰度。
