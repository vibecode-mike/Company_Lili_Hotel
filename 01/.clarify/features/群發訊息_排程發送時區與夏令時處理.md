# 釐清問題

排程發送的日期與時間使用哪個時區？如何處理夏令時調整？

# 定位

Feature：spec/features/create_broadcast.feature Rule 關於排程傳送（約第108-120行）

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 使用台灣時區（UTC+8），不考慮夏令時（台灣無夏令時） |
| B | 使用伺服器時區，需在設定頁面明確顯示時區資訊 |
| C | 支援多時區選擇（未來國際化需求），儲存為 UTC 並轉換顯示 |
| Short | 其他設計（<=5字）|

# 影響範圍

影響排程任務執行邏輯、時間顯示格式、跨時區部署支援，以及未來國際化擴充能力。
