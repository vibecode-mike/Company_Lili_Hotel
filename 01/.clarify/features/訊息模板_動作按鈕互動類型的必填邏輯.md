# 釐清問題

動作按鈕的互動類型（開啟網址/觸發文字/觸發圖片）是「每個按鈕必填」還是「可不設定互動類型」（純裝飾按鈕）？

# 定位

Feature：spec/features/message_template.feature Rule 關於互動類型（約第295-335行）

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 每個按鈕必須設定互動類型，不允許無互動的按鈕 |
| B | 可不設定互動類型，按鈕純裝飾（點擊無反應） |
| C | Template02/03/04 按鈕必填互動，其他模板可選填 |
| Short | 其他規則（<=5字）|

# 影響範圍

影響表單驗證邏輯、Flex Message JSON 生成、按鈕行為設計、使用者期待，以及互動資料完整性。
